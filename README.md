# User Input Example

This is a simple Python program that collects user input for their name, age, and location.
It then prints a personalized message using the provided information.

## How to Run

1. Make sure you have Python installed on your system.
2. Download or clone this repository.
3. Run the script using the command:
   ```bash
   python user_input.py
   ```

4. Follow the prompts to enter your name, age, and location.
